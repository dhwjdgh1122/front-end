반응형 웹 페이지 만들기

화면 크기도 제각각이다. 웹 페이지를 디자인할 때 어떻게 다양한 화면 크기에 대응할 수 있을까?
해결책은 바로 반응형 디자인이다. 화면 해상도에 따라 HTML과 CSS를 자동으로 맞춰서 보여주는 기능을 말한다.

How? 미디어 쿼리를 사용한다 사이트에 접속하는 미디어 타입과 특징에 따라 다른 css스타일을 적용하도록 도와주는 규격이다. 작은 해상도에 맞게 텍스트 크기를 조정하거나, 페이지를 세로 모드로 볼 때 단락 사이의 패딩을 늘리거나, 버튼 크기를 늘릴 수 있다 이렇게 웹 사이트에 접속하는 기기 해상도에 따라서 HTML과CSS가 바뀌는 웹을 "반응형 웹" 이라고 한다.

미디어 쿼리는 다음과 같은 상황에서 사용할 수 있다.
CSS에서 @media어 @import를 사용해 특정 조건에 따라 스타일을 적용할 때
HTML 요소에 media 특성을 사용해 특정 매체만 가리킬 때

미디어 쿼리는 @media로 쓴다 어떤 값에 따라 색깔을 다르게 적용할 때 사용
@import는 다른 스타일시트에서 스타일 규칙을 가져올 때 사용 주의점: 문서 최상단에 있어야한다
코드는 위에서 아래로 해석되고 실행되므로 사용할 파일을 최상단에서 먼저 불러와야 불러온 파일 내용을 이용하는 코드가 동작된다.

전체화면은 흰색이며 너비가 1280px 이하는 다크그레이 색, 600px 이하는 베이직색이다 이유는 style.css에서 @media 미디어타입 (적용 조건)안에 규칙을 사용해 background-color 와 max-width 설정을 해주었기 때문이다.

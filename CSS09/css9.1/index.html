플렉스 박스의 등장

앞에 css08에서는 블록/인라인 레벨 요소로 이미지, 텍스트 등의 요소 위치를 변경해주었다. 과연 블록/인라인 레벨 요소만 알렴ㄴ 모든 요소를 원하는 곳에 위치시킬 수 있을까? 화면 크기가 다양하지 않았던 옛날에는 가능했지만 지금은 더 다양한 레이아웃을 만들고자 고안된 기법이 "플렉스박스"이다.

"유연한"이란 뜻으로 화면 레이아웃을 유연하게, 자유자재로 배치할 수 있게 해준다. 기존에 사용했던 블록 레벨 요소, 인라인 레벨 요소 방식보다 훨씬 강력하고 편리한 기능이 많다.

HTML 코드에서 body를 부모로, 3가지 div를 자식으로 설정하고 CSS 파일에서 div태그에 가로,세로길이,배경색을 지정한다. 또한 div 박스 사이의 구별이 쉽도록 가운데 div 박스에 id 값을 주어 배경색을 초록색으로 했다 div는 블록 레벨 요소이기 때문에 줄바꿈이 된다

맨 위 빨간색 박스에 마우스로 검사도구를 열면 마진값이 오른쪽 투명한 자리가 마진값으로 되어 있다. 이걸 없애려면 플렉스박스를 사용하면된다. 물론 display값을 인라인으로 바꿔도 되긴함

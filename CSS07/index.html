CSS 필요성

CSS: HTML 문서를 시각적을 꾸미는 언어, 왜 HTML에 꾸미기 기능을 넣지 않고 CSS를 별도로 만들었을까 HTML 코드와 꾸미기 코드를 분리하면 가독성과 효율성이 좋아지기 때문! HTML 파일과 CSS 파일을 따로 만드는게 편해서

간단하게 스타일 적용할때 사용하는 방법중에 하나가 HTML에 style 태그를 사용하는거, 주의점은 긴 CSS 코드를 작성할땐 추천 X

style 태그를 사용하면 HTML이 웹 브라우저에게 다음과 같은 말을 전한다. 이 style 태그 안쪽에 있는 코드는 CSS야. CSS 문법에 맞춰 해석하고 처리해줘

style 태그 안에 적용하려는 태그(선택자)를 지정하고 중괄호{}로 둘러싼다 {} 안에 있는 모든 코드는 지정한 선택자 태그에 적용된다

극단적으로 생각해보자. p 태그를 100번 사용한 코드의 색깔을 바꿀려면 100번의 작업을 style 태그를 사용해 반복 작업을 방지할 수 있다

HTML 파일에 head 밑에 link rel stylesheet href = style.css 경로 지정해주고 CSS 파일을 별도로 만들어 태그(선택자)와 {} 를 이용해 꾸미기 기능

CSS 선택자

CSS 선택자는 HTML에 스타일을 적용할 때 HTML 요소를 지정하는 역할을 한다. 의문점,궁금점 "내가 작성한 모든 태그에 스타일을 적용하려면 일일이 태그를 적어야하나?", 또는 "내가 작성한 p 태그 중 첫 p 태그만 CSS를 적용하고 싶으면?" 사고방식 중요 !

전체 선택자

모든 HTML 문서를 선택해 스타일을 적용해준다. 특정 태그 대신 * 기호를 사용한다, head를 포한함 HTML 문서 내의 모든 요소를 선택한다

유형 선택자

특정 태그를 지정해 작성하는 방식 타입 셀렉터라고도 한다, 지정된 태그를 가지는 요소를 사용한다.

아이디 선택자

아이디로 요소를 식별해주는 선택자. 예를 들어 p라는 태그에 아이디를 선언해 해당 선택자에 스타일을 적용한다. 즉 p라는 태그 유형 대신 선언한 아이디로 요소를 선택할 수 있다. 사용 방법 = HTML 에서 p태그를 마감하기 전에 id="아이디" 형식으로 지정한다 "아이디"는 중복 불가능한 고유한 값이어야 한다 = 주민번호, 또 다른 특징: CSS에서 아이디 선택자를 사용하여 스타일링할 때는 #을 아이디 앞에 붙여야 한다.

클래스 선택자

특정 클래스에 속하는 요소를 선택한다. 여기서 클래스란 요소의 목록을 말한다. 아이디와 비슷한 기능을 하지만, 클래스는 여러 태그를 하나의 클래스로 묶을 수도 있고 태그 하나에 여러 클래스 이름을 붙여줄 수도 있다. 예를 들어 p 태그에 클래스 이름으로 c1,c2,c3를 지정해 각각에 스타일링을 지정해놓을 수 있다. 보통은 아이디 선택자보다 클래스 선택자를 많이 사용한다. 하나 이상의 클래스값을 가질 수 있기 때문에 스타일을 주고 싶은 선택자를 자유롭게 지정할 수 있다는 장점. 클래스명 앞에 점.을 붙여주면 된다.

복합 선택자

부모와 자식 태그가 복합적으로 쓰인다 해서 붙여진 이름이라고 생각. 서로의 관계와 위치를 유용하게 결합하는 방식을 제공한다 여기서 '서로'에는 자손 선택자,자식 선택자,인접 형태 선택자,일반 형제 선택자가 속한다. 가장 많이 쓰이는 자손 선택자는 조상의 모든 자손을 선택하는 방법이다. 조상 하위에 있는 자손을 선택한다, 조상과 자손은 띄어쓰기를 통해 구분한다.
